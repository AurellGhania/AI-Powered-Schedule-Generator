# AI-Powered Schedule Generator

Sebuah aplikasi cerdas yang membantu siswa menyusun jadwal belajar secara otomatis menggunakan teknologi AI dari Gemini. Cukup masukkan daftar tugas, durasi, dan deadlineâ€”AI akan mengatur prioritas dan menghasilkan jadwal yang efisien dari pagi hingga malam. Solusi ideal untuk manajemen waktu yang lebih terstruktur dan bebas stres.

# Key Features:
- Menggunakan Gemini API dari Google untuk memproses dan menyusun jadwal.
- Dibangun dengan Flutter agar bisa diakses lintas platform.
- Data tugas dikirim sebagai prompt ke AI, yang kemudian membalas dengan jadwal terstruktur.
